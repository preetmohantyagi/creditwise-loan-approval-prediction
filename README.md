# 💳 CreditWise - Loan Approval Prediction System

An end-to-end Machine Learning project that predicts whether a loan application is likely to be approved based on applicant information. The project covers data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation using multiple classification algorithms.

---

# 📌 Project Overview

Financial institutions evaluate multiple applicant attributes before approving loans. This project automates that process using supervised machine learning techniques.

The workflow includes:

- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis
- Feature Engineering
- Feature Encoding
- Model Training
- Performance Evaluation

---

# 🚀 Features

- End-to-end ML pipeline
- Missing value imputation
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Label Encoding & One-Hot Encoding

# ▶️ Installation

Clone the repository

```bash
git clone https://github.com/preetmohantyagi/creditwise-loan-approval-prediction.git
```

Move into the project

```bash
cd creditwise-loan-approval-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```
